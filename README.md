# Para Tycoon

Bu proje sade bir tarayici oyunu prototipidir.

## Ozellikler

- Isim, sifre ve profil resmi ile giris
- Her saniye otomatik para kazanma
- Elle `+1 TL` toplama butonu
- Saniyelik geliri artiran magaza
- Her satin alimda urun fiyatinin `x2` artmasi

## Nasil kullanilir

1. `index.html` dosyasini tarayicida ac.
2. Yeni hesap olustur veya giris yap.
3. Her saniye gelen parayla magazadan gelir yukseltmeleri al.
